## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bulma-sass'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bulma-sass

## Usage

Import Bulma styles in `app/assets/stylesheets/application.scss`

```css
@import "bulma";
```
